# Cursor Shortcuts Cheatsheet

> Quick Reference: All Important Shortcuts at a Glance

---

## 🎯 Core Shortcuts

### AI Interaction

| Function | Mac | Windows | Description |
|----------|-----|---------|-------------|
| **Inline Edit** | `Cmd+K` | `Ctrl+K` | Inline code generation/modification |
| **Chat Panel** | `Cmd+L` | `Ctrl+L` | Open/close AI chat |
| **Composer** | `Cmd+I` | `Ctrl+I` | Multi-file editing mode |
| **Tab Completion** | `Tab` | `Tab` | Accept AI suggestion |
| **Reject Suggestion** | `Esc` | `Esc` | Reject current suggestion |

### File Navigation

| Function | Mac | Windows | Description |
|----------|-----|---------|-------------|
| **Quick Open** | `Cmd+P` | `Ctrl+P` | Fuzzy search files |
| **Command Palette** | `Cmd+Shift+P` | `Ctrl+Shift+P` | Execute commands |
| **Symbol Search** | `Cmd+Shift+O` | `Ctrl+Shift+O` | Search functions/classes |
| **Global Search** | `Cmd+Shift+F` | `Ctrl+Shift+F` | Search entire project |

## 💡 AI Workflow Shortcuts

### Quick Fix Flow

```
1. Cmd+L        → Open chat
2. Describe issue → AI analyzes
3. Cmd+K        → Apply fix
4. Tab          → Accept suggestion
```

### Feature Development Flow

```
1. Cmd+I        → Open Composer
2. Describe feature → AI plans
3. Preview changes → Check modifications
4. Tab/Enter    → Accept all
```

### Code Optimization Flow

```
1. Select code
2. Cmd+K        → Inline edit
3. "Optimize this code" → AI optimizes
4. Tab          → Accept
```
