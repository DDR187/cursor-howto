# Cursor 快捷键速查表

> 快速参考：所有重要快捷键一览

## 🎯 核心快捷键

### AI 交互

| 功能 | Mac | Windows | 描述 |
|------|-----|---------|------|
| **内联编辑** | `Cmd+K` | `Ctrl+K` | 行内代码生成/修改 |
| **聊天面板** | `Cmd+L` | `Ctrl+L` | 打开/关闭 AI 聊天 |
| **Composer** | `Cmd+I` | `Ctrl+I` | 多文件编辑模式 |
| **Tab 补全** | `Tab` | `Tab` | 接受 AI 建议 |
| **拒绝建议** | `Esc` | `Esc` | 拒绝当前建议 |

### 文件导航

| 功能 | Mac | Windows | 描述 |
|------|-----|---------|------|
| **快速打开** | `Cmd+P` | `Ctrl+P` | 模糊搜索文件 |
| **命令面板** | `Cmd+Shift+P` | `Ctrl+Shift+P` | 执行命令 |
| **符号搜索** | `Cmd+Shift+O` | `Ctrl+Shift+O` | 搜索函数/类 |
| **全局搜索** | `Cmd+Shift+F` | `Ctrl+Shift+F` | 全项目搜索 |

## 💡 AI 工作流快捷键组合

### 快速修复流程

```
1. Cmd+L        → 打开聊天
2. 描述问题      → AI 分析
3. Cmd+K        → 应用修复
4. Tab          → 接受建议
```

### 功能开发流程

```
1. Cmd+I        → 打开 Composer
2. 描述功能      → AI 规划
3. 预览变更      → 检查修改
4. Tab/Enter    → 接受所有
```

### 代码优化流程

```
1. 选中代码
2. Cmd+K        → 内联编辑
3. "优化这段代码" → AI 优化
4. Tab          → 接受
```
